# Isosceles Applet

Iscosceles Applet is a minimal applet demonstrating principles from the ISO7816-4 Smart Card Standard.

## Theory of operation

Upon installation, the Isosceles Applet creates a Master File, a type of Dedicated File.

## Data Object Tags
0x1E
